# ZGU Lab Work - Git

## Описание проекта
Простой проект на Python для демонстрации работы с Git и GitHub. 
Проект создан в рамках выполнения лабораторной работы №2 по информационным технологиям.

## Основные функции
- Вывод приветственного сообщения "Hello from ZGU Polar State University!"
- Сложение двух чисел
- Умножение двух чисел

# Используемые технологии
- Python 3.x
- Git
- GitHub
